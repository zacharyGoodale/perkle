# Perkle Privacy Policy

Effective date: 2026-02-09

Perkle is a private credit card benefit tracking application.

## Scope and Intended Use

Perkle is reserved for private use by invited users. It is not offered as a public consumer service.

## Data We Process

Perkle processes account information and user-provided transaction/benefit tracking data needed to operate the app.

## How Data Is Used

Data is used only to provide core product functionality, including authentication, transaction import, benefit detection/tracking, and user notifications.

## Data Deletion Requests

If you want your data deleted, notify `zach@goodale.dev`.

Deletion requests are handled according to `docs/DATA_RETENTION_AND_DELETION_SOP.md`.
